Exercise 4.1: Draw SVGs

House + garden using SVG primitives + grouped windows.

What shapes I used

  • rect: sky, grass, house body, trunk, door, windows
  • circle: sun, door knob
  • ellipse: tree leaves, shadow
  • polygon: roof
  • line: horizon line
  • path: curved road
  • text: “House” label
